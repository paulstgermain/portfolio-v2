## Getting Started
### Requirements
Before you begin, be sure you have the most recent versions of `node` and `npm` installed.

### Installation Instructions
1. Clone the repo:
```
git clone https://github.com/paulstgermain/portfolio-v2.git
```
2. Install packages:
```
npm install
```

### Usage
Run the app:
```
npm start
```
